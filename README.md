## Getting Started

To get a local copy up and running follow these simple example steps.

### Clone this repository

```
https://github.com/abdulhamiid/microsoftclone.git
```

## Basic Setup
### Install all dependencies

```
npm install
```

### Deployment

Run ```npm start```
